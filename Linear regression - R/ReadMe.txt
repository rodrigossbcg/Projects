# Linear regression - R

In this project we are going to look over linear regression using R. The dataset has only 6 variables. The wfood is the target and represents the proportion of money spent in food by a spanhish family. This project is divided in several parts:

    - Import dataset
    - Data transformation and cleaning
    - Descriptive analysis
    - First Models
    - Managing outliers
    - Models
    - In sample prediction
    - Out of sample prediction
    - Final discussion
    - Prefiction graphs    
    
Note: This project is written in Portuguese, because it was developed within the scope of my degree.
